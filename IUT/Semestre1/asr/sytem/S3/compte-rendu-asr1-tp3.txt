EMACS

1.1 COUPER-COLLER


* Comment déplacer un bloc de texte d'un endroit à un autre ?

->faire CTRL+SPACE, definir la zone de selection et finir pas ECHAP+W
ensuite changé de fenetre avec CTRL+X O, et copier avec la comande CTRL+Y



1.2 AIDE EMACS

* signification de  ctrl-k   

-> "C-k runs the command kill-line" ce qui signifie que ça efface une ligne entiére


* raccourci de "switch-to-buffer" 

-> "switch-to-buffer is an interactive built-in function in `C source code'.
It is bound to C-x b"
la commande de switch-to-buffer est : CTRL+X B



1.3 MACROS

* rôle des raccourcis

ctrl-a : retourner a la ligne

ctrl-f : avancer d'une lettre

ctrl-k : supprimer la ligne entière a partir du curseur

ctrl-y : coller le dernier element copier ou supprimer

ctrl-m : retourner a la ligne

ctrl-u : répéte une comande "n" fois



1.4 UTILISATION DES MACROS

Macro obtenu : 
C-s			;; isearch-forward
SPC			;; next-line
RET			;; debug-help-follow
DEL			;; delete-backward-char
C-u 20 SPC		;; next-line
RET			;; debug-help-follow
<up>			;; previous-line
C-u 20 C-f		;; forward-char
C-k			;; kill-line
<down>			;; next-line
C-a			;; move-beginning-of-line
